### 第十一残页（并联分流，草稿中）

**物理核心：** 并联分流——两条并行路径分担总气势，各路承担一半压力，末端不可相触（否则势聚重蹈焦耳热覆辙）

**古文（草稿，待定稿后冻结）：**
```
水遇岔口，分流而行；
各路之势，不及原流之半，
然合而观之，总势未失。

气之行，与水同理。
路广则势缓，路窄则势涩；
一路分二，各承一半之压，
彼此不扰，此为分路之功。

善用此理者，以弱制猛，
以多路散一势，以细路化急流。
□□勿使两路之末相触，
触则势聚，旧焚之患复至。
```

**顾青旁批（草稿）：**
*「"各承一半之压"——所以两路并走，每路只有一半的势？若引灵石之气，分两路走，是否各只有半数烈度？待试。」*
（第五章触发后修正：「试验成立。"勿使两路之末相触"——此前失败即此因。两路须各自独立，从起至末，无一处相触。」）

### 第卅一残页（动量守恒，草稿中）

**物理核心：** 动量守恒——弃剑坑高重力环境下，坠物冲击力的利用与转化；借重物触发天威做功，而非自己出力

**古文（草稿，待定稿后冻结）：**
```
重者下坠，轻者横飞；
势之行，无强弱之分，
只有方向之别。

物碰物，势不灭，
只换向而行。
重物压轻，重者缓，轻者疾，
各携原势之分，
散于四方。

善用此理者，不以力搏力，
借重物之势，导轻物之向，
以他物之动，换己身之安。

天威愈重，此理愈烈；
坠三尺者，势可当百斤之冲。
```

**顾青旁批（草稿）：**
*「"坠三尺者，势可当百斤之冲"——此坑天威奇重，一尺坠落之势已比别处更猛。可一用。待机。」*
（第六章触发后修正：「用了。重物松动，天威自行做势；小物弹飞，方向难测——此为分势之功。」）

### 第卅八残页（高压锅/筑基，规划中）⭐ 第一卷终点

**物理核心：** 压强→临界相变——高压环境下液体沸点升高，液态灵气可在高压下保持稳定并逐步固化（筑基原理）

**古文方向（骨架）：**

## 六、文体写作规范（新残页创作指南）

> 每次需要为顾青写一段《天工理略》原文时，遵循以下规范。

### 6.1 语言风格

| 维度 | 规范 |
|---|---|
| **句式** | 短句为主，偶有排比。每条"理"用两到三句交代现象，一句给出逻辑 |
| **词汇** | 气、势、路、渊、髓、骨、贯、闭、涩、烈、扬、沉——这类词有质感 |
| **比喻** | 必须用自然意象：水、木、石、风、火、土。禁用工业/现代意象 |
| **避忌** | 禁止出现物理量名词（焦耳、欧姆、帕斯卡等）；禁止现代逻辑连词（因此、所以、即） |
| **缺字** | 适当使用□□，表示损毁。不要每页都有，约三分之一残页有一处缺字即可 |
| **密度** | 每段残页约6-10行，不宜过长，保持"残篇"的碎片感 |

### 6.2 知识获取路径规范

每次顾青引用残页，必须遵循以下四步之一或组合：

```
① 书为根据：呈现古文原文 → 顾青用土著语言重新解读
② 初次误解：第一次读理解偏差 → 实验失败 → 修正
③ 现场触发：看到某个现象 → 联想到书中某句话 → 回去重读
④ 组合推导：书中一句话 + 坑里某个现象 → 推导出应用
```

> 绝不允许顾青"灵光一闪"凭空想到某个原理，必须有书或观察作为根据。
