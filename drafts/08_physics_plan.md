# 第八章格物方案书：东面三十步

## 物理核心

**定律：声速与介质密度（多普勒效应应用）**

### 真实物理原理
声速在密度越高、弹性越大的介质中越快（v = √(B/ρ)，B为体积弹量，ρ为密度）。当叩击产生的声波遇到不同密度的介质界面时，会发生反射。改变叩击频率（快慢）时，对应同一界面的回声延迟变化规律不同：

- 密度均匀的介质：快叩和慢叩的回声延迟比值，等于叩击频率比值
- 前方有更高密度区域：声波在高密度区传播更快，快叩和慢叩的延迟差相对于普通岩壁更小（因为声速快的那段路程比例更高）

实际操作上：叩击从慢到快，观察回声返回时间的变化幅度——幅度越小，说明前方有声速更快的介质，即密度更高的物质。

## 新建残页：第十四残页

**主题归类**：声感探知
**物理对应**：声速与密度——叩击疾徐不同，延迟差揭示前方密度

### 古文正文（待定稿后冻结）

```
击声有疾有徐，疾者入密，徐者入疏；
迟速之差，藏于回声之间。

物之密者，声行疾，来去皆急；
疏者，声行缓，举足即留。

以疾徐互叩，观其迟速之别：
密者，迟速之别小；疏者，迟速之别大。
差若不变，密实无疑；差若倍增，前有空洞。

□□察者，以叩探骨，知前方之理。
```

**作者注**：顾青旁批：
*「快叩慢叩，两者回声快慢之差——差小则密，差大则疏，差若消失，前方必有更密之物。此处之差，比周围缩了太多。往前三十步之内，定有异常。」*

## 知识获取路径

**路径：③ 现场触发 → 联想残页 → 验证推断**

1. **现场异常**：顾青叩壁时，某处的回声比周围来得快，且来得快的程度超出了普通实心岩壁的范围
2. **叩击速度测试**：他改变叩击节奏（慢→快），发现快叩和慢叩的回声时间差，比在普通岩壁上测的要小很多
3. **联想残页**：第十四残页的"以疾徐互叩，观其迟速之别，密者迟速之别小"对应上了
4. **推断**：延迟差比周围小→前方密度比周围高→高品位矿石的可能性

**注意**：顾青的操作是直觉性的——他不是"系统地改变频率然后记录数据"，而是在叩击节奏自然变化时，注意到了某次快叩和慢叩之间回声的不对劲，然后刻意再试了一次确认。

## 声学场景细节

### 叩击工具
顾青用的是他日常工作的铁镐把——这是矿工的标配，叩壁辨声是有经验的矿工都做的事（第十七残页建立了叩石辨髓的基础）。他用拳节叩也可以，用工具叩也可以，本章用工具，更适合控制力度。

### 声音质感描写
- 慢叩普通岩壁：沉，短，在矿道里走不远就散了
- 快叩同一处：略快，但和慢叩的差可以感受到
- 叩到那处异常位置（慢叩）：声音来得比旁边快，质地更硬
- 叩到那处异常位置（快叩）：来得快，但和慢叩的差，比在普通岩壁上要小得多

### 顾青的内心过程
不是计算，是感觉：他叩了一下，感觉不对，停下来，又叩了一下，慢，再叩一下，快。前者来得很快，后者比他预期的更快了——快慢两次的差，比他叩普通岩壁时感觉到的差要小。

他想了一下。往里走，声音是快的，所以差会小——往里走的那段路，声音跑得比普通岩石里快。

前方有比普通围岩更密的东西。

## 三角关系场景设计

### 出场顺序
1. 顾青发现异常（独指翁不在场，他在更深处）
2. 顾青叩出结论时，独指翁刚好走过来（他在东面来回走了三年，路线固定）
3. 顾青说出发现，独指翁的反应（笑点）
4. 炼器弟子苏凌出现（他的采样点就在这段矿道）

### 苏凌（炼器弟子）的登场
- 他进来的时候，顾青和独指翁正在那处异常位置旁边站着
- 苏凌的眼神先扫矿壁，后扫人，他先确认自己的采样位置，再打招呼——他有自己的事，不是为了凑热闹来的
- 他注意到顾青在叩壁那一侧，问了一句：那边叩出什么了？

### 合约谈判场景
苏凌听顾青说出前方有高密度矿石区域，沉默了一会儿，然后说出条件：

"我能让这条消息不到文书那里去。换三成。"

顾青没有立刻回答。他在心里算了一下：没有苏凌的隐瞒，矿脉一旦上报，就是宗门配额项目，他和独指翁分到的是零；有苏凌的隐瞒，顾青和独指翁各拿七成中的一半（三成半），扣除之前算好的开采成本，账面上还是正的。

"行。"顾青说。

独指翁的反应不快，但他没有异议——他找了三年，不是为了把矿脉上报给宗门的。

### 铸器问题种子
苏凌在准备离开时，从腰间解下一件东西，是一截金属管件，接合处有一道裂缝——不宽，但显然不是正常的使用磨损，是反复裂开又反复焊补的痕迹。

"顾青，你看得出这里为什么总裂么？"他把那截管件放在顾青面前，语气随意，像是顺口问的，"我问了炼器科几个师兄，都说是工艺问题，但我改了工艺还是裂。"

顾青接过来，翻了一下，没有回答。

"改日说。"他把管件还给苏凌，"今天没有空。"

苏凌接过去，点了点头，没有追问。

## 独指翁笑点完整设计

顾青用土著语言说出叩击速度疾徐不同、回声之差揭示密度的道理，简短，两三句。

独指翁在原地愣了一下。

他伸手，对着那处异常位置叩了一下，慢。

停了一下，又叩了一下，快。

两次。

然后他把手收回来，站在那里，表情有些奇怪。

"我……我叩了三年。"

"嗯。"顾青说。

"每天叩。"

"嗯。"

独指翁没有继续说了，他的眼神往那处异常位置落了一下，然后往别处看去，好像看的不是矿壁，而是他这三年在这里叩壁的所有那些时候。

顾青没有去说"其实方法不难"或者"以后就知道了"这种话，他只是在那里等着，等独指翁缓过来。
