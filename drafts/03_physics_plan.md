## 二、选用原理

### 地球物理名称
**声学共鸣 / 空腔共振（Acoustic Resonance / Cavity Resonance）**

物理本质：当敲击固体时，固体以自身的固有频率振动并发声。
- **实心致密岩石**：固有频率低，振动能量迅速衰减，声音短促沉闷（"哑声"）
- **含空腔或裂隙的岩层**：空气腔参与共振，固有频率偏高，振动延续，声音明亮清脆（"活声"）
- **矿石嵌合层**：密度与围岩不同，接触面产生声波反射，敲击声有特征性的"二次回响"

**工程验证：** 建筑工程中"叩墙验空"、地质勘探中的浅层地震法，都基于相同原理。

（声学共鸣 / 空腔共振）

### 顾青土著命名
**叩石辨髓** / **虚实听骨**

核心命题（顾青自己的表述）：
> "实者气闭，叩之沉；虚者气通，叩之扬。物之骨髓，藏于声中，非目之所见，乃耳之所辨。"

## 三、《天工理略》知识来源设计

### 3.1 原文段落（残卷第十七残页，字迹有残损）

```
【卷残·器辨第三·第十七页·部分漶漫】

…叩物有声，实者气闭，声短而沉；
空者气贯，声远而扬。

物之内理，藏于击响之中，
非灵目所视，乃□□所辨。

匠人叩墙，知其中实；
□□叩地，知脉所行。

善听者，以耳代目，
以声探骨，以响识髓。□□

（后数行漶漫不可读）
```

> **注**：残损字符用□□表示，顾青无法读出，只能从上下文推断。

### 3.2 顾青第一次读这页（早已读过，但未理解）

顾青在拿到残卷初期就看过这一页。当时他的注解是（记在旁边空白处，用细炭条写的）：

> "叩有声，实沉空扬——已知。但「□□所辨」缺字，不知所指何物。匠人叩墙可验，但矿脉不同于墙，岩层厚密，是否同理未知。存疑。"

他搁置了这条，因为无法确认"□□所辨"是什么。

### 3.3 第三章触发：重读时的新发现

本章顾青停工休息，无意中听到岩壁滴落酸雾时的声音差异（见场景骨架）。他拿出残卷，重读这一页。

**关键误读（第一次假说，错误的）：**

顾青看到"非灵目所视，乃□□所辨"，联系到修仙体系里"灵目"对应的就是"灵耳"——被灵力强化的听觉。他的推断：

> "缺字应是'灵耳'。普通耳朵不够用，需以灵力强化听觉，才能分辨岩层内部的虚实响应。"

**实验与失败：**

顾青将一线灵力引入耳脉（"聪目法"的变体）。结果：灵气波动产生的"嘈杂灵压背景音"把岩石的物理振动完全淹没了。他叩击岩壁，只能感知到灵气流动，什么都听不清。

损耗：0.3斗灵砂（少量），更重要的是留下了术法残迹（虽然极微弱，仍计入风险账本）。

**失败分析（顾青推导过程）：**

顾青看着失败结果，重新检查逻辑链。他注意到原文写的是"以耳代目"——不是"以灵耳代灵目"，是"以耳代目"，是"耳"取代"目"的功能，不是"灵耳"取代"灵目"。

再看"匠人叩墙，知其中实"——匠人没有灵力。

**正确推断：**

> "原文所说的'□□所辨'，缺字不是'灵耳'，是'精耳'或'善耳'——精于分辨的耳朵。这是普通听觉的精准使用，不是灵力强化。书里那个匠人叩墙的例子已经说明了：不需要灵力。"

## 四、应用设计

### 4.1 工具准备
顾青使用的叩击工具：**周虎碎玉**（两钱，质硬，棱角分明——比手指叩击更一致，比矿镐更轻便，不伤手伤）

选用理由（顾青账本逻辑）：
- 碎玉本身已无修复价值（周虎的，两钱，变现也就五分灵砂）
- 用于叩击实验消耗极低
- 小块玉石密度均匀，每次叩击力度更可控，比石块更"量器化"

### 4.2 方法设计（叩石辨髓的操作步骤）

**步骤一：** 在目标岩面划定5×5的叩击矩阵（用炭条在岩壁上轻点定位，事后用手抹去）

**步骤二：** 以碎玉棱角均匀叩击每个节点，用时约一息，等待振动完全衰减后再叩下一点

**步骤三：** 声音分类：
- 沉闷短促（"哑"）→ 实岩，无矿或矿量极少
- 清脆延续（"活"）→ 空腔或裂隙，可能有矿
- 特殊双响（"回"）→ 两种密度层交界，矿石嵌合概率高

**步骤四：** 标记"活"和"回"的节点，优先在这些位置开挖，而非随机挖掘

### 4.3 铺垫要求（本章内，格物高潮前必须出现）

| 铺垫内容 | 出现时机 | 作用 |
|---|---|---|
| 酸雾滴落不同岩面的声音差异（一段明亮一段沉闷）| 停工休息段，自然触发 | 引导顾青注意声音与岩石内部关系 |
| 《天工理略》残卷原文引用 | 停工读书段 | 知识来源 |
| 顾青在旁白边空白处的旧注解 | 读书段 | 表明他早就读过但未理解，学习过程可见 |
| 碎玉道具出现并被顾青拿出来看 | 道具评估段 | 后续用于叩击的工具铺垫 |
| 顾青第一次叩击实验失败（灵耳误读）| 假说验证段 | 科学路径：失败→修正 |

### 4.4 格物场景三要素检验

**① 问题对读者清晰可见：**
顾青的困境：手伤 + 工具坏 + 配额告急，每次挖掘都是盲目消耗体力，效率无法满足缺口。读者能理解这是"找不到矿"的根本问题。✅

**② 解法读者真的猜不到：**
在修仙小说语境下，"用碎玉系统敲石头听声音来找矿"这个解法完全超出预期。读者预期的解法可能是：用灵力探测、找更好的挖矿工具、向其他矿工求助。没有人会想到"声学共振找矿"。✅

**③ 解法揭晓后感觉必然：**
酸雾滴落声的差异铺垫在前；《天工理略》原文"实者声沉，空者声扬"在前；匠人叩墙的比喻在前。读者回看会觉得"这些早就都在了"。✅

## 五、顾青账本核算

**损耗项：**
- 灵力损耗（灵耳尝试失败）：0.3斗灵砂
- 体力损耗（叩击矩阵，约3组，每组25次）：折合体耗0.5日修养量
- 时间损耗：约两个时辰（读书+实验+修正+叩击）
- 碎玉磨损：两钱碎玉，使用后棱角磨损，残值约三分（原值五分）
- 潜在风险：0.3斗的灵力残迹（极微弱，但在执法堂追查周虎期间计提风险）

**收益项：**
- 定位矿脉区域（第二块叩击区出矿）：矿量预估可补配额缺口的六成
- 方法本身的"资产积累"：叩石辨髓可复用，后续每次采矿效率提升（顾青单独记入方法价值账本）
- 第三叩击异常点尚未挖开：可能有更大收益（被崔七出现中断）

**本章净核算：** 微亏（灵力+体耗损耗），但配额缺口从六成改善，方法价值为正。崔七变量未纳入，计入下章风险拨备。

## 六、物理准确性验证

### 声学共振原理是否成立？
✅ 完全成立。空腔共振是基础声学原理：
- 含空腔的岩体固有频率更高，振动时间更长，声音更明亮
- 矿石（金属矿）与围岩密度差异产生声波在界面的反射，有特征性的"二次回响"
- 叩墙验空是建筑工人的实际做法，无需任何先进仪器

### 顾青能在修仙世界发现这个方法吗？
✅ 合理。
- 《天工理略》提供了"实者声沉，空者声扬"的理论基础（匠人经验的抽象）
- 弃剑坑的声学环境特别适合这个方法：酸雾填充空气，减少外部声音干扰；岩层致密，振动传导性好
- 顾青的普通灵根反而是优势：他无法依赖灵力感知，长期锻炼了对物理现象的精细观察

### 可能的误用风险？
⚠️ 顾青目前只能定性判断（"这里有空腔"），不能定量判断（"空腔大小和矿量"）。第一个异常点开挖后是废空腔（旧裂缝），这个失败在方法上是合理的——声音特征区分了"有空腔"和"有矿"，但不同类型的空腔（裂缝vs矿脉）需要更多经验才能区分。这为后续章节的方法升级留下了空间。
