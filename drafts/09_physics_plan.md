# 第九章格物方案书：气路初宽

## 物理核心

**定律：流体阻力正比于路径长度（第七残页"气路流转"原理的升级应用）**

### 真实物理原理（作者知识）
泊肃叶定律：Q = ΔP·r⁴ / (8ηL)
- Q = 流量
- ΔP = 压差
- r = 管径
- η = 粘度
- L = 管道长度

路径长度L增大 → 分母增大 → 流量Q减小。
已知原理"路愈狭，则气行愈涩"（第七残页：r减小→阻力增大）的另一面：
L增大同样能让阻力增大、流量减小。

### 修仙语境映射
- 矿石气息 = 低压强、低流量的气源
- 赤金丝路径 = 传导管道
- 经脉 = 末端容器，有流量上限
- 第一次失败：路径两拃，阻力不足，流量仍超出经脉可承受上限（脉路发涩）
- 修正：路径延长到七拃（绕圈折返），阻力增大三倍，流量降至安全范围
- 第二次成功：气缓慢走入经脉，一个时辰有实质进账

**作者括号注**：`（流体阻力）` — 在顾青想通"路长则势缓，路长一尺，进入的气势就缓一分"时出现。

## 知识来源确认

**本章不引入新残页。**

### 第七残页（升级应用）
顾青已知"路愈狭，则气行愈涩"（焦耳热那章建立的知识）。
本章的新应用：**路愈长，同样行愈涩**。两者都是增大气路阻力的方法，一个是"窄路"，一个是"长路"。顾青在想通这件事时，脑子里会联想到第七残页，但不需要翻开——这是已经内化的知识在新场景的应用。

### 现场观察（热膨胀系数不匹配）
苏凌铸器问题：
- 观察：裂缝沿两种金属的交界线走，干净，不随机
- 推论：两种金属在受热时膨胀幅度不同，冷却后收缩不均，在交界处累积应力，反复使用反复开裂
- 知识来源：顾青在弃剑坑见过铁块和铜块在矿道热气中的反应——铜膨胀比铁快一些，他观察过，不是书里来的
- 解法：焊接时在接合处留余量，给两种金属各自的膨胀余地

## 矿石原位吸收——技术细节

### 矿石气息与灵石气息的差异
| 特征 | 灵石（下品） | 矿石（中品原位） |
|---|---|---|
| 压强 | 高（有明显灵压） | 低（需主动"开路"） |
| 纯度 | 高（灵气集中） | 低（混有矿气杂质） |
| 流量（单位时间） | 高 | 极低 |
| 可持续性 | 一颗用完即止 | 理论上持续，但极慢 |
| 顾青吸收效率 | 因为高压需分流，实际效率不高 | 因为路长阻力足够，效率相对稳定但绝对量极低 |

### 赤金丝路径的物理布置

**第一次（失败）**：
- 一段赤金丝一端贴高密度矿壁，另一端绕到小臂内侧
- 路径直线，约两拃长
- 问题：阻力不足，流量微高，三刻后脉路发涩

**第二次（成功）**：
- 两段赤金丝首尾相接（共约四拃基础长），然后顾青将丝折叠，盘旋，让路径在矿壁和皮肤之间来回走
- 最终实际路径长约七拃（折叠后的有效行程）
- 阻力约为第一次的三倍多
- 流量降到经脉可承受范围，一个时辰后有实质进账

### 物理感官质感

第一次失败的感觉：气息开始走，有点急，像是把一杯水从很近的高处倒下来，不算冲，但也不算缓；三刻后脉路里有一点轻微的燥意，不痛，但说明走得有点快了。

第二次成功的感觉：非常慢，慢到顾青坐在那里半刻钟，都不确定气息有没有在走；一刻后，才确认，确实在走，只是细得像一根线；一个时辰后，他能感觉到脉路里多了一点点，比上次失败时吸到的多，但比吸灵石少太多了。

## 苏凌铸器问题——完整格物方案

### 现象
金属管件，接合处（两种不同金属的焊合处）反复开裂，裂缝沿交界线走，不随机。

### 顾青观察过程
1. 接过管件，翻过来看裂缝走向——沿交界线，不是随机应力
2. 侧对磷光，看两种金属的颜色差——确认是两种成分不同的金属
3. 摸了一下接合处的温度——略高于两侧（摩擦或铸造残余热）
4. 停了一下，想了一下在弃剑坑里见过的铁和铜在热气里膨胀不同的事
5. 结论：两种金属受热膨胀幅度不同，在接合处产生拉力，冷却时收缩幅度也不同，残余应力集中在交界线

### 解法
**在接合处预留一丝间隙**——让两种金属各自有膨胀余地，不强迫它们在热膨胀时贴合。顾青的表达：
"焊接时稍微留一丝缝，让它们各自走各自的。不要让它们挤在一起膨胀。"

### 苏凌的试验
苏凌用一段备用金属试验，最细的夹具调整接合间隙，小炉具加热冷却。

结果：不裂。

### 报酬
灵砂三钱，小布袋，苏凌放在旁边的岩面上说出来。

### 量级说明
三钱灵砂约等于顾青三天月俸，或者说是采矿配额超额奖励的两倍。微薄，但这是"用格物知识帮人"得来的，性质不同。这是全书第一笔特别任务报酬。

## 顾青三钱灵砂的反应——写法设计

**不写成**："他在心里算了一下，三钱灵砂相当于三日月俸。"

**写成**：顾青拿起那个小布袋，感受了一下重量。他在手里多待了一息——那一息里什么都没有想——然后放进了口袋。

然后他说那句："下次有类似的，可以找我。"

这句话本身就是他的反应：他把这个关系记下来了，不只是结束了这笔交易。

## 练气三层——写法说明

**触发时机**：第九章最后三分之一，某日顾青绕过感知丝走出矿道的动作里。

**措辞参考**：
"不知是哪一日，他从矿道入口绕了过去，走出来的时候，脉路里多了一点宽，不多，像是一条细路，被人不知什么时候往旁边挪了半寸。"

**要求**：
- 不是独立段落，和周围的叙事行为连在一起
- 不出现"突破""练气三层""修为提升"等仪式化词汇
- 是顾青感知到的，不是旁白宣告的
- 之后顾青继续做他在做的事，章节不以此结尾

## 结尾钩子设计

苏凌临走前，在离开时说：

"你要是有兴趣，宗门外门有些活是不需要资质的，接了给灵砂的那种。不是采矿，是别的——具体找谁问，我也不清楚，你自己去打听。"

顾青没有回答，苏凌也没等他回答，就走了。

**结尾的场景**：顾青还在矿道里，继续工作，或者在东面矿脉旁边收拾赤金丝路径，苏凌说完那句话走了，矿道又回到了只有叩击声的状态。

**章节结束在一个普通的状态里**——不是什么戏剧性的画面，就是顾青在弃剑坑里继续待着，脉路里多了一点东西，手里多了三钱灵砂，耳边有一句还没消化的话。
