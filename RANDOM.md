# 反馈介绍
看了全部三到八章草稿，第九章和之后还没看。我作为看了一些修仙小说的读者提一些反馈。因为只有我一个人读，所以只代表个人看法，要不要改动，怎么样改动看情况。不要牺牲文章质量大改特改。

# 具体
1、总体文字和设定还是比较扎实的，也看得出来为将来规划了许多。但总体偏沉闷。参考其它的小说，或者以世俗武侠入手，或者宗门外派任务，或者家族文从具体困难入手，总之要从大家比较熟悉的场景开始。这本书一开始就是采矿，一呆就是八章。虽然采矿的确是一个被很多书忽略的部分，也值得写，但没有很多人写在开头，原因就是矿区不是大部分普通读者熟悉的地方。写得再逼真也显得没意思。一般小说出现矿区多半是有些宝物，妖兽，敌人。矿区只是提供地点，有一些景物和人物描写。直接进行大量的具体采矿叙述很难吸引人。三要素中的目标，障碍都不能让读者有情感上的关联。
2、有些话语还是难了些。尽管修改了很多次，以下的段落我没怎么看明白：
   “赤金丝的固定已经稳了，今夜的赶工是值的。他把灵石放在两段丝的衔接处，让它在那里待着，做一块占位石头用——它还是一枚石头，但现在作为灵石，气量只剩了三成，且掺了些围岩矿气的底色，纯度很低，要拿去直接修炼，收益极薄。”（很难想象是什么样的装置，而且这个装置出现在了好几章里）
   “顾青把残卷收进怀里，开始从四周的废料架上取下竹签。不是腕口粗的那种，是加固岩缝用的细竹签，指头细，内腔比根须更均匀。他把那些签一段一段搭在地面的岩缝里，朝着振动传来的方向，把几段本不相连的缝隙接续起来。”（到底是怎么用来挡路的？）

   “矿石截面里，那些细细的红纹不是从一个点长出来的。它们各从不同的位置生出，像树根，各走各的路，彼此之间有间距，向外延伸，只在外壁附近偶尔靠近，但从头到尾没有相触过。。。。
   两段丝，从两个不同侧面的气孔出发，各自独立，中间用竹纤维作隔断，从起点到末端，任何一处都不相触。
   这才是分路。“（这一段是他看到别人的行为产生了启发，应该是爽点桥段，但实在太晦涩了）

   “顾青把两段赤金丝和灵石整体往衬衫内侧推了推，让它们靠紧身体，维持着当前的布置角度，然后往矿渣堆边缘坐过去，把左手放在膝盖上，做出一副在歇息的样子。“（太细致了，现代语旁白混着修仙界的名词，有点奇怪）

   "……你找了多久。"

   "三年多了，还没找到。"

   这次停顿更长。

   "他说的是这一段矿道？"

   "他说弃剑坑东面深处，就这么说，没说具体哪里。"

   监工沉默了一会儿，又问："你一个人在这里？"

   "就我。"又是真话。（这里应该是笑点，完全感觉不出来。对话也和之前的格物描写一样有些压抑）

   “顾青接过管件，翻了一下。裂缝在接合处，不在中间，不在两端——接合处是两种不同颜色的金属，一段是本体，一段是焊补材料，颜色有细微的差别。

   他把管件对着磷光侧过来，看了一下那道裂缝的走向，它是沿着两种金属的交界线走的，干净，不是随机应力的那种断裂。“（太难懂了。大量术语，而且炼器师对材料不可能不熟悉，神识也能扫描材料的问题，这一段本身就比较牵强）


还有一些零零碎碎的现代语的句式。注意很多读者看书修仙小说一开始可能会细致些，一旦没看到什么有意思的桥段之后可能就一目十行了。这种阅读方式对本作的写作风格非常不友好。
也有可能是前三章定下了基调，让执笔感觉一定要顺着这种文风往下写，再加上种种设定，形成了压抑沉闷的氛围，和冗长复杂的描写。反正除了那些修仙名词外绝对看不出这是修仙文。哪怕是一些“真实系”的修仙文也更有想象力。
这其实是硬核科幻+修仙玄幻的调和与平衡。既是本书最大看点，也是最大难点之一。

3、亮点：我感觉能吸引我看得慢一点的地方也有
   # 老矿工出现：
   “顾青把他翻来覆去地算了两遍。

   除掉他：同一处坑底，短期内再死一人，两桩事叠在一起，执法堂的搜查犬不用等人发令就会过来。况且此人身上没有什么可拿的，镐头磨平、衣物破旧，搜出来不够填损耗。白费因果，不合算。“（这是典型的修仙桥段，秘密可能被人发现的紧张感）

   # 新监工出现：
   “顾青第一次感知到新监工，是从岩壁传来的振动，而不是脚步声。
   是那种很细的气息，贴着矿道壁面往前延展，不下压，不冲击，像蛛网一样平铺展开，把前方的每一处岩面都轻轻地探过去。
   周虎从来不用这种方式。周虎的气压是向下的，是猎犬的气压，他一来整条矿道都知道了。这个人不一样，这个人在听，在探，用的是让顾青浑身微紧的那种方式——因为周虎那种气势是可以躲的，而这种网，没有那么容易躲。“（灵压描写，很有修仙味道，角色面对高阶修士时烘托氛围常用。经常让角色对高境界心生向往，既能导向剧情冲突，也能强化主角的目标）

   # 质问：
   然后监工开口，语气没有变化，和来时一样平："明日巳时，去矿道入口处报到。把你见到周虎前后的经过说清楚。"

   "是。"

   "别想着少报一个字。"他说这句话时，眼神不在顾青脸上，在那块已经停在矿道深处的废矿石上。（三要素明确，并且有紧迫感，让读者有共鸣，想着如何解决）

   # 交易：
   苏凌没有立刻说话，他往那段异常矿壁走了几步，伸手叩了一下，听了一下，然后回来了。

   "我能让这条消息不到文书那里去。"他停在顾青面前，语气平，像在谈一件很正常的事，"换三成。"

   顾青把这件事在脑子里转了一下。

   没有苏凌，矿脉若被发现，是宗门配额，他和独指翁分到的是零。有苏凌，他们两个人各拿剩下的七成里一半，开采成本算进去，账面还是正的。他没有能力自己隐瞒这件事——他没有苏凌那个身份，苏凌来这里取样是合法的，他有理由解释自己为什么在这段矿道待着。（经典桥段，修士之间的交易分成。而且直接贴合主角修炼资源的取得，顺其自然）

   # 帮忙：
   独指翁站在原地，等苏凌的脚步声远了，才开口："你看出什么了？"

   "看出一点。"顾青说，"不急。"（应该是主角利用自己的天赋帮别人解决问题的段落，通常是宗门长老后人，受人欢迎的小师妹这类的角色来求助。可惜节奏不好，而且地点也不现实。没有爽快的感觉。）

# 计划
改还是得改，本项目一开始就注定不会好写。只能一点点修正。考虑是不是加一个“读者”AGENT，再加一道步骤。目的翻译暂时用不上，可以去掉或者让项目忽略掉。这个读者应该是看过无数的修仙文，有很丰富的阅读经验和体验，并能做出准确的概括和详细的评价。科幻文和科学知识不是读者必需的，逻辑AGENT可以把关。
然后计划一下怎么从第三章开始修改。另外现在这种审阅方式和仓库的分支方法已经相差很大了。可以说大部分分支已经完全没有意义了。你看情况整合分支吧。不要太复杂。
